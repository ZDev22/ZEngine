A sprite-based C++ Vulkan engine - supports WINDOWS, and maybe LINUX.

Requires C++ installed on your computer to run.

Still in development! Shared so you guys can look at my code!


-----------THIRD-PARTY-DEPENDENCIES---------

GLFW
GLM
VULKAN
EXPRTK
FASTNOISELITE

-------------------COMMANDS-----------------

WINDOWS:
cmake: cmake --preset windows-debug
build: cmake --build --preset windows-debug

LINUX:
cmake: cmake --preset linux-debug
build: cmake --build --preset linux-debug

(Also works for release too)

Get the latest github changes: git fetch origin
